SGNoti는 서강대학교 공지사항을 편리하게 확인할 수 있는 iOS 앱입니다.  
공지사항별 알림과 키워드 알림을 설정하여 새로운 공지사항을 놓치지 않도록 도와줍니다.  

## 둘러보기

* 학사공지
* 일반공지
* 장학공지
* 종합봉사실
* 행사특강
